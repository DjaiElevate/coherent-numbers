## Core Conjecture

Time geometry may not act independently on market behavior. Its effects may depend on the energetic state of the system being observed.

Specifically: the harmonic-calendar signal (if any) that the v0.3.1 MVT is testing may be conditional on, or modulated by, market state variables that the MVT does not currently consider.

A null result on v0.3.1 would disconfirm the unconditional version of the harmonic-calendar claim. It would leave open, but not support, the separate possibility that any effect is state-conditional or energy-dependent. The conditional possibility, if pursued, is a new question requiring its own pre-registration — not a continuation of the v0.3.1 hypothesis.

This is a hypothesis about the structure of any future positive finding, not a rescue argument for a null one. Either way, the v0.3.1 verdict stands as the verdict on the unconditional question.

## Three Distinct Research Threads

The Sadhguru transcript that inspired this note conflates three operationally distinct ideas. They are related but require separate tests:

### Thread A — Time Transformation

Reparameterize the time axis so that one unit of time corresponds to a constant quantity of information, energy, or transactional activity, rather than a constant clock duration.

Operations:
- Volume bars (one bar per N traded shares)
- Dollar bars (one bar per N traded dollars)
- Volatility bars (one bar per N units of realized variance)
- Information-driven bars (Lopez de Prado, "Advances in Financial Machine Learning")
- Intrinsic time (Mandelbrot's multifractal model)
- Subordinated stochastic processes (Clark 1973, Ané & Geman 2000)

Research question: *Does the harmonic-phase signal strengthen, weaken, or persist when the time axis is replaced by an energy-normalized clock?*

### Thread B — Energy Conditioning

Keep clock time, but stratify the data by an energy/state variable and test the harmonic signal within each stratum.

Operations:
- High vs. low realized volatility regimes
- High vs. low volume regimes
- Compression vs. expansion regimes (from Coherent Numbers pullback work)
- Trending vs. mean-reverting regimes
- Pressure states from the two-layer model

Research question: *Does the harmonic-phase signal appear only when the system is in particular energy states?*

### Thread C — Noise Reduction

Keep clock time and keep all states, but filter to a sub-sample of observations where the measurement environment is cleaner (less microstructure noise, less event-driven volatility, less liquidity contamination).

Operations:
- Exclude FOMC days, NFP days, earnings days
- Exclude opening and closing minutes
- Filter to low-bid-ask-spread periods
- Filter to low-news-density periods (sentiment / event-density data)

Research question: *Does the harmonic-phase signal appear only when the measurement environment is sufficiently quiet?*

These three threads are not the same operation. A future research program should test them separately, because conflating them creates an ambiguous interpretation when any of them returns a signal.

The current Coherent Numbers framework holds:

- Layer 1 (universal): pressure dynamics are common across assets

The harmonic-calendar memo (v0.3.1) added one principle:

This is conjectural. It becomes a framework claim only if a Phase 2 study, separately pre-registered, returns supporting evidence.

## Connection to Existing Academic Work

This direction is not fringe. Several established research lines already engage with related questions:

- **Mandelbrot, Multifractal Model of Asset Returns** — local time deformation as a primitive in price-process modeling.
- **Clark (1973), Subordinated Stochastic Processes** — explicit decoupling of clock time from "transaction time."
- **Ané & Geman (2000)** — empirical relationship between volume, volatility, and stochastic clocks.
- **Easley, López de Prado, O'Hara** — information-driven bars and microstructure-aware time discretization.
- **Bollerslev, Tauchen, Zhou** — volatility-of-volatility and state-dependent return predictability.

Positioning a future Phase 2 study within this literature is what separates "extending a serious conversation" from "applying spiritual concepts to markets." The energy/time framing is suggestive; the academic grounding is what makes the work defensible.

## Why This Is Phase 2, Not Phase 1

The v0.3.1 MVT asks one clean question: *does a harmonic coordinate system reveal structure that civil time misses?*

Adding state conditioning or time transformation to that test now would:

1. Multiply the comparison space (multiple states × multiple thresholds × multiple transformations), guaranteeing false positives.
2. Destroy the pre-registration discipline by mid-stream design changes.
3. Make the eventual finding (positive or null) uninterpretable because too many things changed at once.

Phase 1 must finish first. Phase 2 starts from whatever Phase 1 actually shows:

- If Phase 1 returns a clean signal, Phase 2 explores whether the signal strengthens under state conditioning.
- If Phase 1 returns a null, Phase 2 may ask a new, separately framed question about state-conditional effects — but it does not rescue or reinterpret Phase 1.

Either path is informative. Neither requires modifying v0.3.1.

## What Would Make This Note Become a Pre-Registration

Before any Phase 2 test runs, this note would need to be replaced (not amended) by a new pre-registration document that locks:

- Which thread (A, B, or C) is tested first
- The specific operationalization of "energy state" (volatility regime? volume? compression? — must be specified, not chosen later)
- A pre-specified null distribution for the conditional test
- A train/holdout split that is *separate from* the one already used by v0.3.1, OR that uses the v0.3.1 holdout only after explicit acknowledgment that doing so reduces the independence of evidence
- Multiple-comparison correction strategy across threads, if more than one is run

None of this exists yet. None of it should be drafted until v0.3.1 returns its verdict.
