# Coherent Numbers — FAQ

**Is this a new number system?**  
It's a harmonic transform on real numbers with a coherence operator C(·) in base-12 geometry.

**Do you "deny" π being irrational?**  
No. We map it into a finite coherent ratio within a 12-phase octave for applications where harmonic closure is the invariant.

**Why base-12?**  
Divisibility and symmetry (2,3,4,6) yield cleaner phase relationships and closures.
